# План структуры лабораторных работ

## Рекомендуемая структура

### 1. Организация по папкам

```
docs/
├── intro.md                    # Главная страница
├── lab1/                       # Лабораторная работа 1
│   ├── index.md                # Описание и задание
│   ├── theory.md              # Теоретическая часть
│   ├── task.md                # Задание
│   ├── solution.md            # Решение
│   └── assets/                # Изображения, схемы
│       └── diagram.png
├── lab2/
│   └── ...
└── common/                     # Общие материалы
    ├── setup.md               # Настройка окружения
    └── tools.md               # Используемые инструменты
```

### 2. Структура sidebars.ts

```typescript
const sidebars: SidebarsConfig = {
  labsSidebar: [
    'intro',
    {
      type: 'category',
      label: 'Лабораторная работа 1',
      items: [
        'lab1/index',
        'lab1/theory',
        'lab1/task',
        'lab1/solution',
      ],
    },
    {
      type: 'category',
      label: 'Лабораторная работа 2',
      items: [
        'lab2/index',
        // ...
      ],
    },
  ],
};
```

### 3. Шаблон для лабораторной работы

Каждая лабораторная работа должна содержать:

- **index.md** - общее описание, цели, задачи
- **theory.md** - теоретическая база
- **task.md** - конкретное задание
- **solution.md** - решение (опционально, можно скрыть)
