# Методические указания по проектированию арифметического устройства

## Введение

Данные методические указания предназначены для выполнения курсовой работы по проектированию арифметического устройства с использованием языка описания аппаратуры VHDL и пакета Quartus II.

## Цель курсовой работы

Целью курсовой работы является проектирование арифметического устройства для учебного процессора с использованием современных средств автоматизированного проектирования.

## Проекты

В методических указаниях рассматриваются несколько проектов, связанных с проектированием арифметических устройств:
